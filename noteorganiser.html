A Graphical Note Taking software for scientists.

Description

Still in development, it aims to deliver for version 1.0 a fast, useful and cross-platform software to help scientists and professional manage their notes on a daily basis.

Its main features are tag browsing, math support (Mathjax). So far, no currently available software has offered me these needed aspects. The notes are taken with a slightly modified Markdown syntax, so that they remain human readable even when not previewed in html.

Installation

If you have git installed, simply do 
    git clone https://github.com/baudren/noteorganiser
    cd noteorganiser
    python setup.py install --user

Once installed, you can call NoteOrganiser.py from anywhere.

Usage

The first time you call the program, it will create a folder in your home path, called .noteorganiser. You can then place your notebooks there, or create one with the program and start storing your wisdom!

To help you master the modified markdown syntax, please look in the example folder. This contains my accumulated wisdom while developing this application!

Library

Browse your notebooks, organise them in folders.

They are physical markdown files, located in your home directory. You can add and remove notebooks from this interface, and edit them via the Edit panel.

Editing

Edit your notes with a simple text editor.

To create a new entry that respect the modified markdown syntax, simply use the New Entry button.

You can navigate between the different notebooks of the current folder, save the file and reload it from disk.

When you want to browse your notes, push the Preview notebook button.

Preview

Preview your notes in html, with syntax coloring (using Pygments)

Tag filter

The tag list is sorted by number of occurences. By clicking on a tag, it refreshes the preview panel and display only the relevant posts.

The remaining tag buttons available allow for further filtering.
